## 🔫 Quickdraw
<img src="https://img.shields.io/badge/DIFFICULTY-EASY-22c55e?style=for-the-badge&labelColor=1a1a2e"/>

</div>

**Requirement:** Close an Issue or Pull Request within 5 minutes of opening it.

| Step | Action |
|:-:|:--|
| 1 | Open one of your repositories |
| 2 | Go to **Issues → New Issue** |
| 3 | Enter any title and submit |
| 4 | Close it within 5 minutes |

✅ *Also works with a Pull Request.*

## 🦈 Pull Shark
<img src="https://img.shields.io/badge/DIFFICULTY-MEDIUM-eab308?style=for-the-badge&labelColor=1a1a2e"/>

<img src="https://quickchart.io/chart?bkg=white&c={type:'bar',indexAxis:'y',data:{labels:['Default','Bronze','Silver','Gold'],datasets:[{label:'PRs%20Required',data:[2,16,128,1024],backgroundColor:'%2338bdf8'}]},options:{plugins:{legend:{display:false},title:{display:true,text:'Pull%20Shark%20Tiers%20(log%20scale)',font:{size:16}}},scales:{x:{type:'logarithmic'}}}" width="480"/>

**Requirement:** Merge 2 Pull Requests.

| Step | Action |
|:-:|:--|
| 1 | Create a new branch |
| 2 | Make a small change |
| 3 | Commit → push → open a PR |
| 4 | Merge it |
| 5 | Repeat once more |

**Tiers:** ⭐ 2 → 🥉 16 → 🥈 128 → 🥇 1,024

## 👥 Pair Extraordinaire
<img src="https://img.shields.io/badge/DIFFICULTY-HARD-ef4444?style=for-the-badge&labelColor=1a1a2e"/>

</div>

**Requirement:** Be a co-author on a merged Pull Request.

| Step | Action |
|:-:|:--|
| 1 | Create a commit |
| 2 | Add `Co-authored-by: Name <email@example.com>` to the message |
| 3 | Push the changes |
| 4 | Open and merge the PR |

Both contributors receive the badge. **Tiers:** ⭐ 1 → 🥉 10 → 🥈 24 → 🥇 48

## 💡 Growth Tips

</div>

| Tip | Why it matters |
|:--|:--|
| Keep repositories public | Only public activity counts toward badges |
| Contribute to open source regularly | Builds toward Pull Shark & Pair Extraordinaire |
| Participate in Discussions | Fast path to Galaxy Brain |
| Write high-quality documentation | Increases stars → Starstruck |
| Collaborate with other developers | Unlocks co-author opportunities |
| Share your repos on social platforms | Visibility drives stars and sponsors |
